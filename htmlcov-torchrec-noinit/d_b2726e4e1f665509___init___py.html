8"""Torchrec Inference 

9 

10Torchrec inference provides a Torch.Deploy based library for GPU inference. 

11 

12These includes: 

13 - Model packaging in Python 

14 - `PredictModule` and `PredictFactory` are the contracts between the Python model authoring and the C++ model serving. 

15 - `PredictFactoryPackager` can be used to package a PredictFactory class using torch.package. 

16 - Model serving in C++ 

17 - `BatchingQueue` is a generalized config-based request tensor batching implementation. 

18 - `GPUExecutor` handles the forward call into the inference model inside Torch.Deploy. 

19 

20We implemented an example of how to use this library with the TorchRec DLRM model. 

21 - `examples/dlrm/inference/dlrm_packager.py`: this demonstrates how to export the DLRM model as a torch.package. 

22 - `examples/dlrm/inference/dlrm_predict.py`: this shows how to use `PredictModule` and `PredictFactory` based on an existing model.
